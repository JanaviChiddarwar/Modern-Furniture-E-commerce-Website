# Modern-Furniture-E-commerce-Website
A fully responsive and visually appealing Furniture Website UI built using HTML, CSS, and JavaScript. This project showcases modern UI/UX design principles with smooth interactions, elegant layouts, and a clean aesthetic focused on home décor and furniture products.

# 📌 Project Overview

This project represents an e-commerce furniture landing page designed to provide users with an engaging browsing experience. It includes multiple sections such as product collections, categories, testimonials, and a modern footer.

The website emphasizes:

• Minimalist design

• Smooth user interaction

• Clean typography

• Responsive layout

# 🚀 Features

1. Responsive Navigation Bar

    • Toggle menu for smaller screens using JavaScript

    • Smooth open/close animation using class toggling

    • Integrated icons (search, user, cart, menu)

2. Hero Section

    • Large typography with modern styling

    • Highlighted branding text (Innovation)

    • CTA button: Buy Now

    • Product showcase image (sofa)

3. Categories Section

    • Interactive category cards(Seating, Lighting, Tables, Accessories)

    • Hover effects with overlay and icon animation

4. Featured Product (Bed Section)

    • Highlighted product(Description, Material Deatils, Pricing[discounted view using del tag])

    • Glassmorphism-style overlay effect

5. New Arrivals Section

    • Product cards(Image, Category, Title, Price)

    • Hover animation showing “Add to Cart” button

6. About Section
    • Brand introduction (Furnix Homes)

    • Stylish overlapping image design

    • Clean typography and CTA button

7. Parallax Section
    • Background image with fixed attachment

    • Overlay for readability

    • Modern storytelling layout

8. Customer Reviews
    • Testimonial cards(Profile images, Customer names, Feedback)

    • Gradient background styling

9. Services Section
    • Key features(Customer Service, Free Shipping, Secure Payment)

    • Icon-based card layout

10. Footer Section
    • Newsletter subscription input

    • Social media icons

    • Navigation links(Collections, Designers, Information)

    • Contact details

# 🎨 UI/UX Highlights

• Modern color palette using CSS variables:

    - Golden (--golden-hour)

    - Antique brown (--antique-brown)

• Smooth hover and transition effects

• Reusable utility classes (flex, wrapper, gap)

• Clean typography using Google Fonts (Jost)

# 📱 Responsiveness

• Fully responsive design using media queries

• Optimized for:
- Desktop
- Tablet
- Mobile devices

# ✨ Key Design Concepts Used

- Flexbox layout system
- CSS variables for theme consistency
- Overlay effects using pseudo-elements
- Glassmorphism (blur effect)
- Gradient text styling
- Hover-based micro-interactions
